Cryo Gun


The Cryo Gun features two firing modes. The primary firing mode is an automatic machine-gun like mode, then switches to a slower rate of fire when the ammo count reaches 1. The secondary firing mode is a lobby grenade that explodes into smoke clouds when making impact with a surface or player.

The primary firing mode raycasts forward from the camera's location, randomized slightly in a small cone. The ammo check is done every time the gun fires, and when it reaches 1 the firing mode switches the cooldown timer to be slower.

The HUD

HUD Animations play for a variety of events

After the first iteration of the HUD, our team decided to incorporate external feedback that HUD elements looked too bland and unresponsive. It featured no player feedback and didn't feel good when picking up items. It was my responsibility to streamline and liven up the HUD.

HUD animations are done using UMG animations, triggered through gameplay events. UMG and Player Controller blueprints are not replicated over the network so gameplay events must be fired through the Game Mode blueprint.

Working With UMG Animations

Events were fired from the Game Mode blueprint that pickups were grabbed, so the HUD element didn't have to poll for changes. This ensured that the HUD remained responsive and only encapsulated functionality to play the animations.
