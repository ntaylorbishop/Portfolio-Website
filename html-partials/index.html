Master's Thesis: An Exploration in Rendering Pipelines

A 3D rendering engine developed from scratch in C++ using OpenGL 4.5, this thesis was an exploration on modern day rendering techniques. Using the classic Sponza scene for the demonstration, this thesis features many modern-day rendering techniques: Tiled Forward+ rendering, omni-directional shadowmapping, screen-space ambient occlusion, as well as a few others.
